Content Type Data Creator

Written using HTML, CSS, JavaScript

This small script has been written to assist ProfCyberNaught to generate the content data types for pasting underneath the article title, within the article status table in the TODO.md file.

Please select the content data types used within your article (Buttons will disappear once selected, and data types will be listed below):

Below you will see the data you need to add underneath the article title name in the TODO.md article status table (This will create a vertical list below. When copied by clicking the copy data button, the data will be stored as a single line format in your clipboard. Once copied, paste this data between the < sub > tags).
